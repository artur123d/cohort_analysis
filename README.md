# Cohort Analysis

## Описание

В проекте выполнен когортный анализ пользователей с использованием Python.

## Цель

- построить когорты пользователей;
- рассчитать показатель удержания (Retention Rate);
- проанализировать изменение суммы покупок по когортам.

## Используемые инструменты

- Python
- pandas
- Matplotlib
- Seaborn

## Результаты

По итогам анализа построены тепловые карты удержания пользователей и изменения суммы покупок, позволяющие сравнить поведение различных когорт во времени.
